== Description ==

A clean, elegant WordPress theme designed specifically for Gospel Ambition. This theme focuses on simplicity and readability while maintaining a professional appearance. Perfect for ministry websites, blogs, and organizational sites.

== Features ==

* Clean, elegant design
* Fully responsive layout
* Blog functionality with single post templates
* Custom homepage template
* Search functionality
* SEO optimized
* Fast loading
* No comments system (as per requirements)
* Custom post navigation
* Related posts functionality
* Mobile-friendly navigation
* Google Fonts integration (Inter)
* Customizable hero section

== Installation ==

1. Upload the theme files to the `/wp-content/themes/gospel-ambition` directory
2. Activate the theme through the 'Appearance > Themes' menu in WordPress
3. Customize the theme through the WordPress Customizer
4. Set up your menus in 'Appearance > Menus'
5. Configure your homepage settings in 'Settings > Reading'

== Theme Structure ==

The theme includes the following templates:
* front-page.php - Homepage template
* index.php - Blog index page
* single.php - Single blog post template
* page.php - Static page template
* search.php - Search results template
* 404.php - Error page template
* header.php - Site header
* footer.php - Site footer

== Blog Setup ==

To set up the blog functionality:
1. Create a new page called "Blog" 
2. Go to Settings > Reading
3. Set "Your homepage displays" to "A static page"
4. Choose your front page and set "Posts page" to your "Blog" page
5. Blog posts will now be accessible at /blog/ and individual posts at /blog/[post-name]

== Customization ==

The theme includes custom meta boxes for the homepage hero section. You can customize:
* Hero title
* Hero subtitle  
* Hero description
* Button text and link
